````
Alguns conceitos:

- next = é framework, o mesmo pega essa estrutura criada por react, usa o react-dom para renderizar isso na HTML e depois adciona varias funcionalidades a esses conjuntos;
- react = é abstração da nossa interface, mas não é uma interface renderizada na tela ainda;
- JSX = parece ser, mas é apenas semelhante ao HTML, pois é uma extensão de sintaxe para JavaScript. Recomendamos usar JSX com o React que fornece uma maneira de estruturar a renderização de componentes usando uma sintaxe familiar para muitos desenvolvedores; 
- Pagina Estatica = não pega dados em lugar nenhum e não muda;
- <Link> </Link> => tem que importar do next/link;
````
